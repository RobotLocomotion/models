# Schunk WSG50 Hydroelastic Bubble Meshes

## Gripper Base

The files
```
./meshes/bubble_gripper_base.mtl
./meshes/bubble_gripper_base.obj
```
were created in CAD using Schunk's drawing in the [product catalog](https://schunk.com/fileadmin/pim/docs/IM0026091.PDF) (WSG 050-110-P model). Please see `LICENSE` for more information.

## Punyo bubble fingers

The files
```
./meshes/bubble_finger.mtl
./meshes/bubble_finger.obj
./meshes/ellipsoid_bubble_geometry.mtl
./meshes/ellipsoid_bubble_geometry.obj
```
are based on: https://punyo.tech/#instructions

## Disclaimer 

These are very fine visual meshes that are not intended to be used as collision models without seeing a considerable slow-down in simulation speed.
